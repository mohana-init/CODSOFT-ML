# CODSOFT-ML
# Iris Flower Classification 

## Objective

This project classifies iris flowers into different species using Machine Learning.

## Algorithm Used

* K-Nearest Neighbors (KNN)

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn

## Features

* Dataset inspection
* Data visualization
* Model training
* Prediction
* Accuracy evaluation

## Output

The model predicts iris flower species such as:

* Setosa
* Versicolor
* Virginica

## Conclusion

The KNN model successfully classified iris flowers with high accuracy using flower measurement features.
